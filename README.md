# AYOBA MTN REWARDS

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/demidovahelenka00-ctrl/mtn-rewards.git
   ```
2. Navigate to the project directory:
   ```bash
   cd mtn-rewards
   ```
3. Install the necessary dependencies (if applicable):
   ```bash
   npm install
   ```

## Features
- User registration and login.
- Rewards points system.
- Ability to redeem points for rewards.
- User-friendly interface.
- Mobile-friendly design.

## Deployment Guide for GitHub Pages
1. In your repository, go to **Settings** > **Pages**.
2. Under **Source**, select the branch you want to deploy (e.g., `main`) and the folder (usually `root` or `/docs`).
3. Click **Save**.
4. Your site will be published at `https://<username>.github.io/mtn-rewards/` after a few minutes.
